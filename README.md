# PaperAssistant - 智能论文助手

> 基于 HelloAgents 框架的 AI 学术研究助手，帮助研究者快速完成论文检索、阅读理解、总结分析以及引用生成。

## 📝 项目简介

PaperAssistant 是一个面向科研人员和学生的智能论文助手。

### 一、解决什么问题？
传统科研文献阅读流程：手动搜索论文→下载PDF→逐篇阅读全文→人工总结核心内容→手动排版整理参考文献，整体流程繁琐、耗时长、效率低下，还容易出现文献检索不全、论文理解不透彻、引用格式出错等问题，增加科研前期文献调研的成本。
本项目基于大语言模型Agent技术，自动化完成全流程文献调研工作，大幅减少人工重复劳动，提升文献研读和报告撰写效率。

### 二、特色功能
整体由 **Research Agent 主智能体** 统筹调度3个核心模块，形成端到端工作流：
1. **文献检索工具**：根据用户科研问题，精准检索、筛选、汇总相关学术文献，完成文献收集
2. **论文总结Agent**：批量精读论文内容，自动提炼研究背景、方法、核心结论、不足与创新点，完成结构化内容解读
3. **引用生成Agent**：自动按照指定学术格式生成规范参考文献
4. 最终自动整合输出完整的**结构化科研报告**，直接响应用户原始科研问题

### 三、适用场景
- 学生课程论文、毕业论文开题调研、文献综述撰写
- 科研人员课题前期调研、方向梳理、快速精读大量外文文献
- 课程学术汇报、科研周报/月报撰写、课题立项前期资料整理
- 学术写作初稿素材整理、文献引用规范化校对
- 入门研究者快速学习领域前沿研究进展

### 四、整体流程
用户提出科研问题 → Research Agent 调度执行文献检索、论文精读总结、规范引用生成 → 产出结构化科研报告，实现科研文献调研全流程智能化。

## ✨ 核心功能

- [智能文献检索] 功能1:输入研究主题后自动分析关键词，搜索相关论文，返回论文标题、作者、年份。
- [AI论文总结] 功能2:支持生成论文背景，核心方法，实验结果，创新点，局限性。
- [自动生成论文引用] 功能3:自动按照指定学术格式生成规范参考文献，支持BibTeX格式，APA格式，IEEE格式。
- [多Agent协作] 功能4:系统包含三个智能体：Research Agent文献搜索，Summary Agent论文总结，Citation Agent引用生成，实现全流程自动化。

## 🎯 项目亮点

- 亮点1:依托大模型 Agent 实现自动任务规划，可将用户研究目标拆解为检索、筛选、总结、引用全流程步骤，自动完成文献调研工作。
