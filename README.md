# PaperAssistant - 智能论文助手

> 基于 HelloAgents 框架的 AI 学术研究助手，帮助研究者快速完成论文检索、阅读理解、总结分析、引用生成及 PDF 下载。

---

## 📝 项目简介

PaperAssistant 是一个面向科研人员和学生的智能论文助手，**通过大语言模型 Agent 自动化完成文献调研全流程**，大幅减少人工重复劳动，提升科研效率。

### 传统痛点
- 手动搜索、筛选论文 → 耗时
- 逐篇阅读全文 → 难以快速获取核心信息
- 手动整理摘要、引用 → 容易出错
- 文献管理杂乱 → 后续查找困难

### 解决方案
用户只需输入研究主题（支持中文），程序自动：
1. 检索 arXiv 相关论文
2. 自动翻译中文关键词（若需要）
3. 批量生成结构化总结（背景、方法、创新点、实验、局限性）
4. 生成规范 BibTeX 引用
5. **按需下载 PDF** 并按主题分类保存
6. 自动合并增量结果（已有论文不足时自动补充）
7. 输出 JSON/Markdown 报告，并生成索引汇总

---

## ✨ 核心功能

- **智能文献检索**  
  支持关键词、年份、分类筛选，自动将中文主题翻译为英文，提高搜索精准度。

- **AI 论文总结**  
  调用大模型 Agent 自动生成结构化摘要，包括：研究背景、方法、创新点、实验结果、局限性。

- **自动生成引用**  
  支持 BibTeX 格式，自动提取 arXiv ID、作者、标题等信息，输出规范引用条目。

- **PDF 自动下载**  
  一键下载论文 PDF，并按主题分目录保存，方便本地管理。

- **智能增量补充**  
  同一主题多次运行时，自动检查已有结果数量；若不足期望篇数，则**自动搜索并排除已有论文**，只补充缺失部分，避免重复处理。

- **批量处理**  
  支持一次性输入多个主题，全自动逐一处理，适合批量文献调研。

- **双重缓存加速**  
  - **搜索结果缓存**：避免重复请求 arXiv，加速二次查询。
  - **LLM 响应缓存**：相同论文再次处理时直接读取缓存，节省 API 费用和时间。

- **结构化输出**  
  生成 JSON 数据（便于二次分析）和 Markdown 报告（便于阅读），并自动生成 `outputs/index.md` 汇总所有主题的结果文件链接。

## 🚀 快速开始

### 环境要求
- Python 3.11 及以下版本（推荐 3.11，避免`cgi` 模块兼容问题）
- 一个支持 OpenAI API 兼容接口的大模型 API Key（如 DeepSeek、OpenAI 等）

# 克隆项目
git clone https://github.com/xiangfeifei65-code/PaperAssistant.git
cd PaperAssistant

# 创建虚拟环境（可选）
python3.11 -m venv .venv
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt

### 配置API密钥

# 创建.env文件
cp .env.example .env

# 编辑.env文件，填入你的API密钥

### 运行项目

python main.py

## 📖 使用示例

请输入要搜索的论文主题（例如：Agent Memory）：光电子
请输入要处理的论文数量（默认 3）：3
是否将结果保存到 outputs/ 目录？(y/n，默认 n)：y
是否尝试下载论文 PDF？(y/n，默认 n)：y

程序会自动搜索、总结、生成引用，并下载 PDF 到 data/pdfs/光电子/，结果保存到 outputs/光电子/光电子.json 和 outputs/光电子/光电子.md。

## 🎯 项目亮点

- 全自动化流程：从检索到报告生成，无需人工干预。
- 智能增量更新：重复处理同一主题时，只补充缺失的论文，避免重复劳动。
- 双重缓存机制：大幅提升重复查询速度，节省 API 费用。
- 批量处理能力：一次性处理多个研究主题，适合文献综述和课题调研。
- PDF 自动下载：论文 PDF 按主题分类保存，方便本地阅读。
- 结构化输出：JSON + Markdown 双格式，兼顾机器可读和人类可读。
